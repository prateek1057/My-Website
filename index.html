Headings

HTML headings are defined with <h1> tags where <h1> defines the most important heading and <h6> defines the least important heading.

Paragraph

Paragraphs are also main parts of an HTML page after headings.<p> tag defines a paragraph. In this tag browser removes the extra spaces and line brakes
<pre> tag defines a performatted text.The text inside a <pre> element is displayed in a fixed-width font (usually Courier),and it preserves both spaces and line breaks

Style

style is an attribute in a HTML page.style attribute has th following syntax:
<tagname style="property:value">
There are 5 properties in this attribute:

  • color
  • background-color
  • font-size
  • font-family
  • text-align

Color property sets color in the text in which this property is applied.
Background-color property sets color in the background of HTML page.
font-size property adjusts the size of that particular text on which this property is applied
font-family property changes the type of that particular text
text-align property changes the alignment of that particular text.

Comments

Comments are used to increase the readability of an HTML code.
Comments are defined as <!--    (text)   -->

Links

HTML links are used to link any other web page in a HTML page.
Links has following syntax:
< a href="   (URL of web page)   ">

Images

Image element is used to add an image in a HTML page.we can also adjust the height and width of an image.The image element has this following syntax:
<img src="   (adress of image)    "   height="   "   width="   ">

Classes

The HTML class attribute is used to define equal styles for elements with the same class name.So, all HTML elements with the same class attribute will get the same style.Different elements can have same classes.
An example to define a class: <h1 class="   (Class name)   ">

Id

The id attribute specifies a unique id for an HTML element (the value must be unique within the HTML document).The id value can be used by CSS and JavaScript to perform certain tasks for the element with the specific id value.An Id is defined for a unique element.
An example to define an Id: <h1 id="   (Id name)   ">

iframe

An iframe is used to display a web page within a web page.We can also set style attribute and height and width of an iframe.An HTML iframe is defined with the <iframe> tag:
<   iframe src="URL"  height="  "  width="  "> </iframe>

Some more important points about HTML

  • <br> tag is used for line brake in HTML page.

  • &nbsp is used for extra space in paragraph.

  • &gt is used for right angular bracekt in HTML page.
